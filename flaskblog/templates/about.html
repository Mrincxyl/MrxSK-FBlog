About This Blog

Welcome to Flask Blog — a simple blogging platform built with Flask. It allows users to create accounts, write posts, update their profile, and share ideas with others.

✨ Features

  • User authentication (register, login, logout, password reset)
  • Create, edit, and delete posts
  • Profile management with profile pictures
  • Responsive and clean design

👨‍💻 About the Developer

This project was developed as a practice project using Flask. It combines Python, SQLAlchemy, and Flask extensions like Flask-Login and Flask-WTF to demonstrate how a full-stack web app can be built.

The goal is to keep learning, improving, and building more advanced projects 🚀.
